# cst0400_web_cw1 — Lily (static e‑commerce)

## Description
A small, static e‑commerce website for a fictional natural‑supplement brand **Lily**. Built with plain HTML and CSS for the CST0400 coursework. The site is fully client‑side (no JavaScript) and demonstrates layout, navigation, basic forms, media embedding and responsive styling.

## Pages & key features
- `index.html` — Home: brand intro, value points and product image.
- `products.html` — Product listing: three product sections with static forms, a promotional video and a pricing table.
- `contact.html` — Contact details: opening hours (table + list), contact info, social links and an embedded Google Map iframe.

Other notable features:
- Navigation header present on all pages
- CSS in `CSS/style.css` (grid layout, responsive-friendly blocks)
- Forms are static (no backend); product quantity inputs submit to the same page
- Media references: `Images/product.png` and `Images/adsvd.MP4` are referenced by pages

## How to preview 
- Open `HTML/index.html` in a browser. 

## File structure (source) 
- HTML/

  - `contact.html` — Contact & map
